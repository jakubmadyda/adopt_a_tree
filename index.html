Spacing

Use variables or classes for spacing.

Spacing helpers can be used for padding, margins, positioning elements, etc.

The spacing variables are $spacing, $spacing-2, $spacing-3 and $spacing-half.

The default $spacing variable's value is 10 pixels.

$spacing-2 is twice the value of $spacing and so on.

Reference files: _globals.scss, _spacing.scss, _variables.scss

Icons

Adopt-a-Tree's font icon set was created with the Icomoon app. It includes font icon families like Feather, Font Awesome, Icomoon and Typicons.

Each icon has a class with a prefix of at-icon-.

To update the icon set, click the 'Import Icons' button on the Icomoon website and upload the selection.json file from the font-icon folder inside the assets folder. Add or delete icons as needed. Download the new icon set, copy the contents of the CSS file into the font-icons partial and update the paths in the @font-face rule in the file.

Font icons which are purely decorative can be hidden from screen users using the aria-hidden attribute.
