Spacing

Use variables or classes for spacing.

Spacing helpers can be used for padding, margins, positioning elements, etc.

The spacing variables are $spacing, $spacing-2, $spacing-3 and $spacing-half.

The default $spacing variable's value is 10 pixels.

$spacing-2 is twice the value of $spacing and so on.

Reference files: _globals.scss, _spacing.scss, _variables.scss

Icons

Adopt-a-Tree's font icon set was created with the Icomoon app. It includes font icon families like Feather, Font Awesome, Icomoon and Typicons.

Each icon has a class with a prefix of at-icon-.

To update the icon set, click the 'Import Icons' button on the Icomoon website and upload the selection.json file from the font-icon folder inside the assets folder. Add or delete icons as needed. Download the new icon set, copy the contents of the CSS file into the font-icons partial and update the paths in the @font-face rule in the file.

Font icons which are purely decorative can be hidden from screen users using the aria-hidden attribute.

Buttons

Add button classes to anchors, buttons or any appropriate HTML element.

If you must use elements like div and span for buttons, add an attribute of role="button" for screen readers.

Buttons can be of different colors.

Buttons can be of different sizes (default, small, extra small and full-width).

Buttons can be solid or hollow.

Buttons can have disabled styles attached to them.

Buttons can have their styles reset so they no longer resemble buttons.

Each button uses a base btn class and modifier classes.

To create a new solid or hollow button of a different color, create a new modifier class and call the btn-solid mixin or btn-hollow mixin. Include arguments as needed.

Reference files: _buttons.scss, _variables.scss

Forms

Wrap form elements in a form tag.

Use the field class to add a bottom margin to form elements.

Text-based inputs, textareas and selects use the form-text class for styling and the same classes for sizes, variants and states.

Inverse styling classes are also available.

Ensure the for attribute of a label matches the ID attribute of the input associated with it.

Reference files: _forms.scss, _variables.scss

Grid

To create a grid structure, start with a container, include a row and add columns. Use flex helpers with rows or float helpers with columns.

A 12 column grid is provided with classes for each media query breakpoint. E.g. col-12, col-6-sm, col-4-mid and col-3-lg.

To change the number of columns, change the value of the grid-column-count variable in the variables partial. Doing so will automatically create new columns across all breakpoints. For example, you could change the numbers of columns from 6 to 12 or 24.

Reference files: _grid.scss, _variables.scss

Grid: Containers

Container sizes are obtained from the media query variables.

Each container is horizontally centered and has left and right padding.

A full-width container also exists for sections that span the entire width of the screen.
